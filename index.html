Gilbert Johnson Keerthi Distance in R^2

Github Repository

The Gilbert Johnson Keerthi (GJK) minimizes the separation between two convex sets. You can use this to either determine if arbitrary convex polygons are intersecting, or iteratively approximate when they will intersect along a given direction.
